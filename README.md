## Features

 * Live preview for markdown.
 * Post/update text posts.
 * Fetch exist posts.
 * Multi blog support.

### Limitation

 * Support only text posts.

## Download/Install

 1. Download from https://github.com/mzp/tumblotte/releases.
 2. Open `Tumblotte-YYYYMMDD.dmg`
 3. Drag `Tumblotte.app` to Applications.

## For Developer

### Build

```
npm install
npm run gulp build
npm run electron
```

### :package:

```
npm run gulp dmg
```
