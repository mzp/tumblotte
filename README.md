## Features

 * Live preview for markdown.
 * Post/update text posts.
 * Fetch exist posts.
 * Multi blog support.

### Limitation

 * Support only text posts.

## Build
### Prepare

```
npm install
```

### For debug

```
npm run gulp build
npm run electron
```

### :package:

```
npm run gulp dmg
```
